/synthetic.png

This applet generates synthetics. It's definitely a bit buggy.

Quick URL:

https://agile.geosci.ai/synthetic.png

Example URL with the default values:

https://agile.geosci.ai/synthetic.png?imps=0,1,0,-1,0&thicks=4,2,4,2,4&polarity=normal&freq=med&phase=0&cmap=RdBu

The parameters (defaults in bold):

  • imps — the relative impedances (unitless) of the model layers, from the top down — e.g. 0,1,0 (a hard layer) or 0,1,0,-1,0.
  • thicks — the relative thicknesses (unitless) of the layers — e.g. 4,2,4 (a thin layer) or 4,2,4,2,4.
  • polarity — normal (US, SEG) or reverse (European) polarity — normal or reverse.
  • noise — noise or not — 0 (no noise) or 1 (add some power-law noise).
  • freq — the frequency of the wavelet used — lo, med, vhi, or hi. Spellings 'low' and 'high' are allowed.
  • phase — the phase of the wavelet used — integer, degrees, default 0.
  • cmap — only matplotlib colourmaps are allowed, see matplotlib help. E.g. RdBu, seismic, RdGy, gray. You can reverse any colourmap by appending _r, e.g. gray_r.

The synthetics are matplotlib plots generated by a Python Flask web app running on pythonanywhere.com.
