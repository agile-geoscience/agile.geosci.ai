/bruges.png

This applet generates a non-deterministic 'logo' for bruges, a geophysics library. "Bruges" stands for "Bag of really useful geophysical equations and stuff", but you can perturb the words bag, really, useful, and stuff.

Example URL with the default values:

https://app.scienxlab.org/bruges.png?n=4&p=0.5&style=none

The parameters:

  • n — the number of words that can be perturbed, takes 1, 2, 3, or 4 (all of them).
  • p — the probability that the name will be perturned, takes values 0 (never) to 1 (always).
  • style — bubble or none (or anything else).
